Description

NeLter is computer tool designed to help amatuer players improve at No Limit Texas Hold 'em. It features a full fledged simulation game where the user can test their abilities against any number of computer opponents that employ a range analysis algorithm and basic strategy to mimic a real life opponent.

Installation

This project requires python3, pillow, and requests to be installed in order to run. Python3 can be downloaded here. Once downloaded run the following code for your respective operating system in a python file, and run the output in a terminal or command prompt instance.

MacOSX/Linux: '''python import sys print(f'sudo "{sys.executable}" -m pip install pillow') print(f'sudo "{sys.executable}" -m pip install requests') '''

Windows: import sys print(f'"{sys.executable}" -m pip install pillow') print(f'"{sys.executable}" -m pip install requests')

Now to start the application run the nelter.py file with python.
